Detailed Description

A classe Map representa o tabuleiro do jogo, este é representado por uma matriz de duas dimensões, e representa o espaço no qual o robô se desloca para coletar joias e energia. Esta classe armazena todas as informações dos elementos do mapa.

void Coletor_Joias.Map.tabuleiroProximaFase ( Map  m)

Este método posiciona todos os elementos das fases 2 em diante no mapa, as joias, obstáculos e o robô. A partir da segunda fase, os elementos são posicionados de maneira aleatória no mapa, portanto foi analisada a proporção de cada elemento na fase inicial e esta mesma proporção de quantidade foi replicada para os elementos de cada tipo. Este método aumenta as dimensões do mapa em 1 unidade a cada nova fase do jogo. Este método também imprime na tela uma mensagem quando o jogador passa de fase.
