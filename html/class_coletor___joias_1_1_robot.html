Detailed Description

A classe Robot é uma especialização da classe Cell. Ela representa o robô que se movimenta pelo mapa coletando joias e energia e desviando dos obstáculos.

◆ baixo()

void Coletor_Joias.Robot.baixo ( Map  m,
Robot  r 
)

O método verifica a posição atual do robô, verifica se há algum objeto instransponível abaixo dele e se não houver um objeto instransponível, ele realiza a movimentação para baixo. Este método também verifica se há algum objeto do tipo Radioactive nas posições adjacentes ao robô quando este se movimenta para baixo, se houver são tirados 10 pontos de energia do robô.

Parameters
m
r

Verifico a posição atual do robô no mapa

Verifico se não há nenhum elemento intransponível no local para qual o robô irá se movimentar.

Se a posição que o robô irá sobrepor for um elemento radioativo, o robô perde 30 pontos de energia.

Insiro "--" na posição atual do robô.

Para onde o robô vai, nova posição do robô no tabuleiro.

Inclusão do robô em sua nova posição no tabuleiro.

Tira energia caso seja realizada a movimentação.

Quando ele tentar ultrapassar os limites do mapa ou tentar se movimentar para a posição que tem uma joia ou obstáculo, será lançada uma exceção informando que o movimento não é permitido.
